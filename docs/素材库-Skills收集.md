## 一、职业发展 Career

### 1. 简历精修 ⚡🌐
- **What：** 上传简历 → AI分析+逐条修改建议+生成优化版本
- **竞品：** CoverDoc、Prentus（简历+模拟面试+追踪器全套）
- **价值：** 需要跑模型，用户必须回来，结果导向明确
- **验证：** 牛客AI面试、智联AI简历分析已有市场

### 2. 谈薪模拟 ⚡🌐
- **What：** AI扮演HR/老板，用户练习谈薪资、谈晋升、谈绩效
- **竞品：** CoverDoc（谈薪建议嵌入job guide）
- **价值：** 对话互动，粘性高，实时反馈

### 3. 面试模拟 ⚡🌐🇨🇳
- **What：** AI扮演面试官，实时提问→用户回答→AI反馈+评分
- **竞品：** Prentus（最完整）、牛客AI面试、Hone
- **价值：** 强烈需求，付费意愿中-高

### 4. 职业规划顾问 ⚡🌐
- **What：** AI职业顾问对话，评估技能→给出发展方向→制定行动计划
- **竞品：** Noomii AI Coaching、Purpose.app
- **价值：** 追踪进度，长期陪伴，用户粘性高

### 5. 求职追踪管理 📝
- **What：** AI辅助管理求职进度，提醒跟进，自动化跟进邮件
- **竞品：** Prentus（all-in-one）

## 关键发现

1. **海外竞品多但国内空白**：情绪心理、面试模拟、谈薪等场景海外已跑通，国内几乎无专注中文场景的产品
2. **AI加工服务 > 纯Prompt**：需要跑模型的场景用户粘性高，广告价值大；纯Prompt库容易被ChatGPT替代
3. **情绪心理市场最成熟**：$1.88B市场，51%症状改善数据，付费意愿已验证
4. **简历/面试是流量入口**：强需求，年轻人高频，SEO容易起量
5. **生活决策场景小红书验证过**：租房、谈薪、买房分析等内容互动极高，证明需求真实存在
